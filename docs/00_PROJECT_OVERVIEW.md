# Project Overview

## Project Name
Kamojan Laundry Website

## Vision
Membangun website resmi Kamojan Laundry sebagai aset digital jangka panjang yang meningkatkan kepercayaan pelanggan, memperluas jangkauan pemasaran, dan menjadi sumber utama akuisisi pelanggan melalui pencarian Google dan WhatsApp.

## Objectives
- Membangun website company profile yang modern dan cepat.
- Mengoptimalkan Local SEO untuk wilayah Desa Cimahi, Kecamatan Cicantayan, Kabupaten Sukabumi.
- Menyediakan informasi layanan, harga, lokasi, jam operasional, dan kontak secara jelas.
- Mengarahkan seluruh konversi menuju WhatsApp.

## Target Audience
- Mahasiswa kos.
- Warga sekitar.
- Pelanggan baru yang mencari layanan laundry melalui Google.

## Scope
In Scope:
- Landing page.
- Halaman layanan.
- Halaman harga.
- Halaman tentang.
- FAQ.
- Kontak.
- Blog.

Out of Scope:
- Dashboard.
- Login.
- Membership.
- Tracking cucian.
- Pembayaran online.

## Technology
- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide Icons
- Vercel

## Documentation Structure
Seluruh spesifikasi proyek berada di dalam folder docs dan harus dibaca berurutan sesuai AGENTS.md.

## Development Philosophy
Business First → SEO First → Content First → Design System → UI Specification → Technical Implementation.
