# SEO Specification

## Objective
Maksimalkan visibilitas Kamojan Laundry pada hasil pencarian Google, khususnya Local SEO untuk wilayah Desa Cimahi, Kecamatan Cicantayan, Kabupaten Sukabumi.

## SEO Principles
- SEO First Architecture
- Semantic HTML
- Mobile First
- Fast Loading
- Helpful Content

## Metadata
Setiap halaman wajib memiliki:
- Unique title
- Meta description
- Canonical URL
- Open Graph
- Twitter Card

## Structured Data
Gunakan JSON-LD untuk:
- LocalBusiness
- BreadcrumbList
- WebSite

## URL Structure
/
/about
/services
/pricing
/faq
/contact
/blog
/blog/[slug]

## Technical SEO
- XML Sitemap
- robots.txt
- Canonical URL
- Optimized images
- Lazy loading
- WebP/AVIF

## Content SEO
- Satu keyword utama per halaman.
- Heading hierarchy H1-H3.
- Internal linking.
- CTA menuju WhatsApp.

## Performance Targets
- Lighthouse Performance >= 95
- Accessibility = 100
- Best Practices = 100
- SEO = 100

## Local SEO
- Konsisten menampilkan nama bisnis, alamat, nomor WhatsApp, jam operasional, dan Google Maps pada halaman kontak serta footer.
