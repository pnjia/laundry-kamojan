# Design System

## Purpose
Dokumen ini mengadaptasi DESIGN.md sebagai fondasi Design System Kamojan Laundry. Struktur, typography, spacing, token, dan pola dokumentasi dipertahankan, sedangkan identitas visual disesuaikan dengan karakter bisnis laundry.

## Design Principles
- Content First
- Consistency
- Accessibility
- Performance
- Simplicity

## Brand Personality
- Bersih
- Ramah
- Modern
- Terpercaya
- Lokal

## Color Direction
Primary: Blue
Secondary: Sky Blue
Accent: Warm Yellow
Surface: Off White
Success: Green
Warning: Amber
Error: Red

## Typography
Menggunakan spesifikasi typography pada DESIGN.md tanpa perubahan hierarchy.

## Layout
- 12-column desktop
- 8-column tablet
- 4-column mobile
- Mobile-first

## Components
Komponen mengikuti DESIGN.md dengan adaptasi konteks bisnis:
- Button
- Card
- Navbar
- Footer
- Hero
- Service Card
- Pricing Card
- FAQ Accordion
- Gallery

## Iconography
Gunakan outline icons (Lucide).

## Photography
Prioritaskan foto asli Kamojan Laundry dibanding stock photo.

## Motion
Animasi ringan:
- Fade In
- Hover
- Accordion
- Scroll Reveal

## Accessibility
- WCAG compliant
- Kontras memadai
- Focus state jelas
- Touch target minimum 44px
